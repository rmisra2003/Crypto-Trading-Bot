

# 🤖 Crypto Trading Bot

An automated algorithmic trading bot that executes cryptocurrency trades based on technical indicators and real-time market data. Designed to help traders make data-driven decisions and reduce emotional trading.

---

## 🚀 Features

* 📈 **Automated Trading** — Executes buy/sell orders based on predefined strategies
* ⚡ **Real-Time Market Data** — Fetches live price data from exchange APIs
* 🧠 **Technical Indicators** — Supports indicators like RSI, MACD, Moving Averages
* 🛡️ **Risk Management** — Includes stop-loss and take-profit mechanisms
* 🔄 **Strategy-Based Execution** — Modular logic for testing different trading strategies

---

## 🏗️ Tech Stack

* **Language:** JavaScript / TypeScript
* **Runtime:** Node.js
* **APIs:** Crypto Exchange APIs (e.g., Binance, etc.)
* **Libraries:** Axios / Fetch (for API calls), Technical Indicators libs

---

## 📊 How It Works

1. Fetches real-time market data from exchange APIs
2. Applies technical indicators to analyze trends
3. Generates buy/sell signals based on strategy logic
4. Executes trades automatically via API
5. Manages risk using stop-loss and take-profit rules

## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/rmisra2003/Crypto-Trading-Bot.git

# Navigate into the project
cd Crypto-Trading-Bot

# Install dependencies
npm install
```

---

## 🔑 Setup

Create a `.env` file in the root directory and add your API keys:

```env
API_KEY=your_api_key_here
API_SECRET=your_api_secret_here
```

⚠️ Never share your API keys publicly.

---

## ▶️ Running the Bot

```bash
npm start
```
